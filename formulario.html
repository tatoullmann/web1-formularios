¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

HTTP define un conjunto de métodos de petición para indicar la acción que se desea realizar para un recurso determinado.

ejemplos: GET, HEAD, POST, PUT, DELETE, CONNECT, OPTIONS, TRACE,PATCH.
